# Expense Tracker (Трекер расходов)

## Автор
[Тимофеенко Софья]

## Описание программы
Expense Tracker - это десктопное приложение для отслеживания личных расходов. Программа позволяет вести учет финансов, категоризировать расходы, фильтровать записи и анализировать траты за выбранный период.

### Основные возможности:
- Добавление расходов с указанием суммы, категории и даты
- Фильтрация расходов по категории и периоду
- Автоматический подсчет общей суммы расходов
- Сохранение данных в формате JSON
- Удобный графический интерфейс
- Валидация вводимых данных

### Категории расходов:
- Еда
- Транспорт
- Развлечения
- Здоровье
- Образование
- Коммунальные услуги
- Одежда
- Другое

## Требования
- Python 3.6 или выше
- Библиотека tkinter (входит в стандартную поставку Python)
